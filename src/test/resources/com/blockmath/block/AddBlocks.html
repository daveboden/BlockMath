Can place block type whole_fraction at (6, 40, 7)

The whole fraction block is 60 units high which equates to 60 slabs or 30 whole blocks in Minecraft.

Stone obstruction stops the block being placed

If a stone obstruction is placed at (6, 60, 7) then a block type whole_fraction can no longer be placed at (6, 40, 7)
